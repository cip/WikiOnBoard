WikiOnBoard by cip

Offline reader for zim files, in particular the wikipedia, for mobile devices. It is based on QT and zimlib. It is cross-platform and supports Symbian, Maemo, Linux and Windows based devices.

Dependencies

QT 4.6.2, zimlib, liblzma

Install

Symbian: Download http://github.com/downloads/cip/WikiOnBoard/wikionboard_installer.sisx and install on your mobile phone. This file includes the Nokia Smart Installer, which downloads and install QT if required before installing the WikiOnBoard App. If QT is already installed you can directly install http://github.com/downloads/cip/WikiOnBoard/wikionboard.sisx. For other OS no binaries are available yet.
